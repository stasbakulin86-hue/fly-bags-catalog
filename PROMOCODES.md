# Промокоды FLY-BAGS

Промокоды хранятся в файле `promocodes.json`.

## Готовые примеры

| Промокод | Что делает |
|---|---|
| FLY300 | Скидка 300 ₽ на любой чемодан от 2 990 ₽ |
| FLY500 | Скидка 500 ₽ на чемоданы M и L |
| SET1000 | Скидка 1 000 ₽ на наборы |
| SALE10 | Скидка 10% на товары от 4 000 ₽ |

## Как отключить промокод

Поменяйте:

```json
"active": true
```

на:

```json
"active": false
```

## Какие ограничения поддерживаются

Можно ограничить промокод по:

- минимальной цене: `minPrice`;
- размеру: `sizes`;
- категории: `categories`;
- материалу: `materials`;
- цвету: `colors`;
- конкретным артикулам: `products`.

Пример промокода только на один артикул:

```json
{
  "code": "BLUE500",
  "title": "Скидка 500 ₽ на синий чемодан M",
  "type": "fixed",
  "value": 500,
  "products": ["FB-M-BLU"],
  "active": true
}
```

## Важное ограничение

Так как это бесплатный статичный сайт на GitHub Pages, промокоды не являются секретными и не списываются автоматически. Покупатель видит цену со скидкой, а продавец подтверждает скидку вручную.
