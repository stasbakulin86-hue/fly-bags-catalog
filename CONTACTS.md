# Контакты на сайте

В файл `index.html` добавлен блок `contacts-section` с двумя карточками магазинов.

Данные по умолчанию:

- Воронеж: Московский проспект, 58. Режим: ежедневно 10:00–19:00.
- Липецк: ул. Валентины Терешковой, 18. Режим: ежедневно 11:00–19:00.
- ВКонтакте: https://vk.com/chemodan_lipetsk

Если нужно заменить ссылку ВК, найдите в `index.html` строку:

```html
href="https://vk.com/chemodan_lipetsk"
```

и замените адрес на нужную ссылку.
